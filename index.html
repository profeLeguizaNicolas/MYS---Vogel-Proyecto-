Sistema Interactivo – Método Vogel

Objetivo

Desarrollar una aplicación que permita al usuario ingresar datos de un problema de transporte y obtener automáticamente la solución utilizando el método Vogel.

Modalidad de Trabajo

Grupos de hasta 4 integrantes.

Parte 1: Desarrollo del Software

Funcionalidades obligatorias

  • Ingreso de datos:
    • Cantidad de orígenes
    • Cantidad de destinos
    • Matriz de costos
    • Oferta de cada origen
    • Demanda de cada destino
  • Validaciones:
    • Verificar que la oferta total sea igual a la demanda total
    • Mostrar mensaje de error si no se cumple
  • Resolución automática mediante el método de la esquina noroeste
  • Visualización de resultados:
    • Matriz de asignaciones
    • Costo total del transporte

Tecnologías sugeridas

  • HTML + JavaScript (recomendado)
  • Excel
  • C++

Requisitos del Software

  • Debe ser completamente funcional
  • Permitir múltiples ejecuciones
  • Interfaz clara y ordenada
  • Código organizado

Parte 2: Informe

Estructura del informe Obligatoria

  1. Portada
    • Datos institucionales
    • Integrantes
    • Fecha
  2. Introducción
    • Breve explicación del problema de transporte
    • Objetivo del sistema desarrollado
  3. Desarrollo del Software
    • Tecnologías utilizadas
    • Explicación del funcionamiento del programa
    • Estructura del sistema
  4. Ejemplo de uso
    • Tutorial de como utilizar el sistema
    • Paso a paso de como utilizar el Software
  5. Conclusión
    • Experiencia del grupo
    • Dificultades
    • Posibles mejoras

Entrega

  • Software funcionando
  • Informe completo

Criterios de Evaluación

  • Funcionamiento del sistema
  • Interacción con el usuario
  • Correcta aplicación del método
  • Claridad del informe
  • Organización del código


Evaluación Alternativa.

Los estudiantes podrán optar por rendir una evaluación individual de manera práctica con el método de resolución Vogel.

Aquellos estudiantes que aprueben dicha evaluación con una calificación igual o superior a 7 quedarán eximidos de la presentación.
